# createdAt

本作《createdAt:》は、電子ペーパーを表示装置とした装置統合型のビジュアル生成作品である。
電子ペーパーを表示装置とする生成システムは、処理を実行した場所・時間・実行者といった環境情報を元にGenerative Artを提示する。提示される作品は画像データ等としては生成・保存されず、電子ペーパー上に表示された瞬間に初めて視覚情報として結実する。デジタル作品が本質的に抱える複製可能性に対し、作品の生成と提示を装置と一体化することでデジタルアートに一回性と固有性を内包させる。電子ペーパーの出力特性や、変数を可視化する補助装置を含めたシステム全体を1つの作品として扱うことで、デジタル手法を利用しながらも生成プロセスや環境によって作品に文脈と物質性を与える試みた研究である。
