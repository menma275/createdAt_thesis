# createdAt

《createdAt:》は、電子ペーパーを表示装置とした装置統合型のジェネラティブアートである。
実行する時刻・場所・実行者といった環境情報を変数としたビジュアルを生成し表示する。 このビジュアルは、電子ペーパーに表示された瞬間に初めて視覚情報として結実するもので、システム内部では画像データを持たず保存されることがない。作品生成と提示装置を一体化したことで、その瞬間限りの固有のビジュアルを生み出している。従来の複製可能なデジタル作品のあり方から離れ、生成の一回性と作品の唯一性を追求した。
本作品ではシステム全体を1つの作品として扱い、計算装置・生成プログラム・変数を可視化する装置・画像を無電力で保持できる電子ペーパーから構成する。 デジタル手法を利用しながらも生成プロセスや環境によって作品に文脈と物質性を与えることを試みた。
